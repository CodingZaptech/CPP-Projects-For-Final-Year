# Currency Converter (C++ Project)

This is a beginner-friendly Currency Converter project in C++. It helps you convert amounts between USD, INR, and EUR using fixed exchange rates. The program runs on the console and is easy to understand for learners.

---

## Features
- Convert **USD ↔ INR**
- Convert **USD ↔ EUR**
- Simple console-based interface

---

## Required Technologies
- **C++ programming language**  
- **I/O Streams** (`cin`, `cout`)  
- **Standard Library** (`iostream`, `iomanip`, switch case)  

---

## How to Run
g++ CurrencyConverter.cpp -o currency_converter
./currency_converter

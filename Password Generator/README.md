# Password Generator (C++ Project)

## Introduction
This Password Generator creates a secure random password of user-defined length. It also logs each generated password to a file for reference.  

## Features
- Generate random passwords with letters, numbers, and symbols  
- Save generated passwords  
- Simple console interface  

## Required Technologies
- **C++ programming language**  
- **I/O Streams** (`cin`, `cout`, `ofstream`)  
- **Standard Library** (`string`, `cstdlib`, `ctime`)  

## How to Run
1. Save as `PasswordGenerator.cpp`  
2. Compile: g++ PasswordGenerator.cpp -o passwordGenerator
